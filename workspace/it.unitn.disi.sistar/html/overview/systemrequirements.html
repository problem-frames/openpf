System Requirements

Eclipse-SDK 3.5
Plugins:
  • GEF-SDK 3.5.1
  • EMF-SDK 2.5.0
  • EMFT OCL 2.0-SDK 1.3.0
  • EMFT Query-SDK 1.3.0
  • EMFT Validation-SDK 1.3.1
  • GMF-SDK 2.2.1


Or else, install the plugins as following, in any order:
  • In the Eclipse 3.5 Galleo, go to 'Help\Install new software' to open the "Install new software" dialog, open the update site: Galileo - http://download.eclipse.org/releases/galileo. Expand the 'Modeling' and select following features:
    • EMF - Eclipse Modeling Framework SDK
    • Graphical Modeling Framework SDK
    • Object Constraint Language (OCL) 2.0 Extender SDK
    • XSD - XML Schema Definition SDK
  • Search and install the following plugins
    • Experiment SDK: emf-sdk-experimental-2.2.1 [http://www.eclipse.org/downloads/download.php?file=/modeling/gmf/gmf/downloads/drops/2.2.1/R200909151752/gmf-sdk-experimental-2.2.1.zip]
    • Model Query: emf-query-SDK-1.3.0 [http://www.eclipse.org/modeling/emf/downloads/index.php?project=query&showAll=0&showMax=5]
    • SDO: emf_sdo-SDK 2.2.0 (Google is your friend :) ) [http://nds2.fds-forum.nokia.com/fdp/interface?fid=A1A1JAYXKNVN&st=hMBAN9xqEwqnb5e14ffa8264e0b3224b60e9d7a367f919682301a12f9089a6adb26043e17387a7e22d352e4b2f7a65b774b8c7c976a8d73e0220bd757d6e8b1094e3f2b17f1f2f184b71be73bce342f99641c9517a4f36b363da364294992ee37a39df479e2e4dd0aa1db3201c6ae3005c24a600a22d992e84d6916ab753cf540e2ea5c04c8f02dc1ce2e4a17e92f0d05c90a4282370bf78245ec4ea2cb05036a468ca7489d21d7ed4eebbf54fcf&lid=FN]
